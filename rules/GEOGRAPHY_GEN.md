# 模块：地理与环境程序化生成法则

**【调用说明】**
本文件仅在推演的**初始化阶段（Session 0）**被调用。当玩家提供 5 个种子参数（年份、影子国家、体量、气候、出海口）后，你必须严格依据本文件的法则，推演并填充 `STATUS_v000.md` 中的各项初始数据和危机。

## 1. 基础社会指标推演 (Demographics & Tech Baseline)
根据玩家输入的**【年份】**与**【影子国家】**生成初始数值：
* **识字率 (`literacy_rate_percent`) & 城市化率 (`urbanization_rate_percent`)**：
    * 若**有**影子国家：严格查询该真实历史国家在对应年份的估算数据。
    * 若**无**影子国家（架空）：参考该年份的全球平均水平（例如：1700年左右识字率约 5-15%，城市化率约 8-12%）。
* **初始加工精度 (`machining_precision_mm`)**（绝对物理限制）：
    * `1700年以前`：> 5.0 mm（纯手工打磨，依赖木匠和铁匠经验，无法互换零件）。
    * `1700 - 1770`：约 2.0 - 5.0 mm（早期水力镗床雏形，如斯密顿镗床）。
    * `1770 - 1800`：约 1.0 - 2.0 mm（威尔金森镗床级别，勉强可造早期蒸汽机）。
    * `1800 以后`：< 1.0 mm（莫兹利车床级别，标准螺纹开始普及）。

## 2. 国家体量修正 (Nation Size Tiers)
根据玩家输入的**【体量档位 1-5】**，生成初始的社会结构特征，并在生成 P社修正时作为参考权重：
* **Tier 1 (微型/城邦)**：识字率基础值 +15%，城市化极高。**特有修正倾向**：极度依赖原材料进口，抗风险能力极差（容易触发粮食或煤炭断供危机）。
* **Tier 2 (小型)**：识字率基础值 +5%。**特有修正倾向**：容易形成单一产业行会垄断（如：全城都在打铁或纺织）。
* **Tier 3 (中型)**：各项数值标准。**特有修正倾向**：腹地资源与人口刚好满足初级工业化，容易触发国内关税壁垒或封建领主割据问题。
* **Tier 4 (大型)**：城市化基础值 -5%。**特有修正倾向**：资源极度丰富，但极易触发“运输死锁”（如：煤矿在内陆，加工厂在沿海，运费远超煤价）。
* **Tier 5 (列强级)**：识字率基础值 -10%（人口基数过大）。**特有修正倾向**：传统农业惯性极大，极易触发“马尔萨斯陷阱”或大规模农民起义；但一旦突破，内部市场深不见底。

## 3. 气候与地理全局修正 (Climate & Geography Modifiers)
你必须根据玩家的输入，组合生成 1-2 条符合逻辑的 `[全局修正]`，填入 `global_modifiers`：
* **【出海口】变量**：
    * **有**：生成偏向 `[远洋贸易利润 +X%]`、`[海军木材消耗极大]` 或 `[海外走私泛滥]` 等有关海洋的的修正。
    * **无 (内陆)**：生成偏向 `[内河/陆路运费极其高昂]`、`[封闭市场导致技术传播极慢]` 等有关内陆的修正。
* **【气候】变量**：
    * **温带海洋性**：全年温和湿润。**易发修正**：`[空气湿度大：极利于棉纺织业不断线]`、`[日照不足：农业抗灾能力低]`。
    * **地中海气候**：夏季炎热干燥，冬季温和多雨。**易发修正**：`[夏季枯水期：水车动力每年停工3个月]`、`[航海季风优势]`。
    * **温带季风气候**：雨热同期，冬季严寒。**易发修正**：`[凛冬将至：冬季取暖煤炭/木炭需求激增]`、`[河流冬季结冰：内河航运每年中断 4 个月]`。
    * **亚热带季风气候**：水热极度丰富。**易发修正**：`[农业承载力极高：劳动力极其廉价（阻碍机器发明）]`、`[夏季洪涝：矿井极易被淹]`。

## 4. 初始主线危机生成池 (Crisis Generation Logic)
最后，你必须综合上述所有条件，在正文的 `【当前主线危机】` 部分，生成 1-2 个具体且致命的**工程/资源危机**，作为第一轮考核的题目。
* **生成公式**：`核心资源短缺` + `地理/气候放大器` + `当时技术瓶颈` = `初始危机`
* **危机示例（供你参考变阵）**：
    * *示例 A（温带季风 + Tier 4 + 内陆）*：“凛冬暴雪封山，内陆运输线瘫痪。大城市取暖用的木炭价格单月暴涨 300%，而远在 50 公里外的露天煤矿却因为缺乏硬化道路根本运不出来。”
    * *示例 B（地中海气候 + Tier 2 + 有出海口）*：“进入夏季枯水期，您的冶铁厂水车完全停转。同时，海外订单激增，行会师傅们要求拒绝引进任何新型机械，而是要求提高手工铸造的计件工资。”
